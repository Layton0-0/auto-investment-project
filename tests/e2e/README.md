# E2E 테스트 (루트 tests/e2e)

이 폴더는 **AI 팀 테스트 구조**의 E2E 분류용입니다.

- **실제 Playwright E2E** 위치: `investment-frontend/tests/` (playwright.config.ts 기준)
- 실행: `cd investment-frontend && npm run e2e` 또는 `npx playwright test`
- 전체 QA: `.\scripts\run-full-qa.ps1` (E2E 단계 포함)

Cursor QA Agent는 `investment-frontend/tests/` 에 E2E 테스트를 생성합니다.
