# Security

## Reporting a vulnerability

Please use [GitHub Security Advisories](https://github.com/Layton0-0/auto-investment-project/security/advisories/new) for this repository. Do **not** file a public issue for an undisclosed vulnerability.

Include enough detail to reproduce or assess impact. Avoid pasting live secrets, API keys, or production credentials in the report.

## Scope

This advisory entrypoint covers the **parent monorepo** (`auto-investment-project`). Submodule repositories (`investment-backend`, `investment-frontend`, etc.) have their own GitHub projects; use each repo’s Security tab when the issue is specific to that codebase.

## Response

Maintainers will acknowledge valid reports as capacity allows. This is a best-effort open source project; SLA is not guaranteed.
