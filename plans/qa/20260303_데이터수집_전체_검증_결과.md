## 4. 재검증 시 사용한 명령 (PowerShell)

```powershell
# 헬스
Invoke-RestMethod -Uri "http://localhost:8001/health" -Method GET

# 뉴스/공시 수집 (응답: received, saved)
Invoke-RestMethod -Uri "http://localhost:8001/dart-collect" -Method POST
Invoke-RestMethod -Uri "http://localhost:8001/sec-collect" -Method POST
Invoke-RestMethod -Uri "http://localhost:8001/yonhap-collect" -Method POST
Invoke-RestMethod -Uri "http://localhost:8001/naver-collect" -Method POST
Invoke-RestMethod -Uri "http://localhost:8001/google-news-collect" -Method POST

# US 일봉 (거래일 지정)
$body = '{"bas_dt":"2026-02-27","symbols":["AAPL","MSFT"]}'
Invoke-RestMethod -Uri "http://localhost:8001/us-daily" -Method POST -Body $body -ContentType "application/json"
```
